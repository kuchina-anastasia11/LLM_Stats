# Использование LLM для выявления вхождений статистических тестов в научных статьях и интерпретации их результатов

НИУ ВШЭ Кучина Анастасия Романовна 

Методы и средства принятие решений на основе проверки статистических гипотез широко применяются a во всех областях науки. При этом разнообразие применяемых тестов и форм представления и описания их результатов затрудняют их автоматическое выделение в текстах статей. Требуется исследовать возможности применения больших языковых моделей для данной задачи.
